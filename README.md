## Installation

To get a local copy up and running, follow these steps:

1. **Clone the repository:**

```sh
git clone https://github.com/roufhasan/cosmic-digital.git
```

2. **Install dependencies**

```sh
cd cosmic-digital
npm install
```

3. **Start the server**

```sh
npm run dev
```
